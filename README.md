## Clone repo
```bash
git clone https://github.com/huzefarana/yt-video-downloader/
```
## Change directory
```base
cd yt-video-downloader/
```
## Create virtual env
```bash
python -m venv venv
```
## Activate venv
```bash
source venv/bin/activate
```
## Install requirements.txt
```bash
pip install -r requirements.txt
```
# Run app
```bash
python main.py
```
